# marketing-skills-collection

> Generate marketing deliverables across CRO, copywriting, SEO, analytics, and growth using 23 sub-skills.

## 23 Sub-Skills
**CRO**: page-cro, signup-flow, onboarding, form-optimization, paywall
**Copy**: copywriting, copy-editing, email-sequence, social-content
**SEO**: seo-audit, programmatic-seo, comparison-page, schema-markup
**Ads/Analytics**: analytics-tracking, paid-ads, ab-test
**Strategy/Growth**: launch-strategy, pricing-strategy, retention, churn-analysis, growth-experiments, referral-program, content-strategy

## Instructions
**S1: Define Objective**
- Single KPI (conversion, CTR, activation)
- Target audience + segment
- Channel and format
- Offer + positioning

**S2: Select Sub-Skill**
Pick closest match from 23 options

**S3: Build Prompt**
- Product summary + pain points
- Offer and differentiation
- Tone/voice constraints
- Output format

**S4: Generate & Validate**
- Generate asset
- Validate KPI alignment
- Revise with specific feedback

**S5: Handoff**
- Implementation checklist
- Tracking events + thresholds
- A/B test proposals

## Best practices
1. One KPI per deliverable
2. Audience specificity
3. Instrument measurement
4. Iterate with data

## Validation (Multi-Agent)
- R1 (Orchestrator): Category coverage
- R2 (Analyst): KPI alignment
- R3 (Executor): Actionability check
